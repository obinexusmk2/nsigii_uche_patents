\section{Classical Foundation}

\begin{definition}[Classical Force Field]
A force field is a vector field
\[
\mathbf{F}: \mathbb{R}^n \to \mathbb{R}^n
\]
that assigns to every point $\mathbf{r} \in \mathbb{R}^n$ a vector $\mathbf{F}(\mathbf{r})$ representing the force experienced by a test particle located at that point.
\end{definition}

If $\mathbf{F}(\mathbf{r})$ derives from a scalar potential $\phi(\mathbf{r})$, then
\[
\mathbf{F}(\mathbf{r}) = -\nabla\phi(\mathbf{r}),
\]
and the field is called \emph{conservative}.
